# Trading Data Directory

This directory stores CSV files with trade logs.

## Files Generated:
- `livetrading_YYYY-MM-DD.csv` - Daily trade records
- Backtest result Excel files

## Structure:
```
Date, Time, Instrument, Entry, Exit, Profit, Strategy, Status
```

⚠️ **Note**: CSV files are excluded from Git (.gitignore) to keep repository clean.
